Pack de composants *bouchons* pour Vitanéo
==========================================

Dossier à copier dans votre projet à l'emplacement: src/components

Contenu:
  - contact-form.tsx
  - price-table.tsx
  - mini-simulator.tsx
  - sla-banner-full.tsx
  - csv-download-button.tsx
  - configurator-grid.tsx
  - configurator-toolbar.tsx

Ces composants sont simplifiés pour éviter les erreurs 'Module not found' et
permettre la navigation. Remplacez-les ensuite par vos vrais composants.
